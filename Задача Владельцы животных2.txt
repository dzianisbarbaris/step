Задача: Владельцы животных (Owner) и их питомцы

Добавьте в программу новый класс Owner (владелец животного), который будет содержать информацию о владельцах домашних животных (Pet).

Требования к классу Owner

Поля:
 • private int id - уникальный идентификатор
 • private String name – имя владельца.
 • private String surName – фамилия владельца.
 • private static String[] names - массив имен.
 • private static String[] surNames - массив фамилий.
 • private Pet[] pets - массив животных которые есть у владельца.
 • private static int totalOwners – общее количество владельцев.

Методы:
 1. Статический блок (static):
 • Должен выводить сообщение "Класс Owner загружен в память!" при первой загрузке.
 • Должен инициализировать массив имен и фамилий. По 5 штук достаточно.
 2. Конструктор: (без аргументов)
 • При создании нового Owner увеличивать totalOwners.
 • Рандомно выбираем фамилию и имя. 
 
 3. Геттеры и сеттеры для name, surName, id и pets
 4. Метод getTotalOwners() – возвращает количество созданных владельцев.
 5. Метод добавить addPet() - который добавляет в массив питомца и добавляет текущего owner к питомцу.

Доработки в Pet
 1. Добавить поле Owner owner. Соответсвенно удалить ownerSurName ownerName
 2. Добавить методы:
 • setOwner(Owner owner) – устанавливает владельца.
 • getOwnerName() – возвращает имя владельца или "Без владельца", если владелец не установлен.

Доработки в Zoo (main)
 1. Создайте нескольких владельцев (Owner).
 2. Создайте несколько домашних животных (Pet).
 3. Назначьте владельцев питомцам.
 4. Выведите информацию о каждом питомце, включая владельца.
 5. Выведите общее количество владельцев (Owner.getTotalOwners()).
