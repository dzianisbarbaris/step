Задача: Библиотека книг
Все поля должны быть привытные доступ к полям через get и set.
Основные классы

 1. Создайте класс Author, который содержит:
 • Поля: name (имя), surname (фамилия), birthYear (год рождения).
 • Конструктор для инициализации полей.
 • Метод toString(), который возвращает строку вида "Фамилия имя - год рождения".
 
 2. Создайте класс Book, который содержит:
 • Поля: title (название книги), author (автор), year (год издания).
 • Конструктор для инициализации полей. И пустой констуруктор.
 • Метод toString(), который возвращает строку вида "Название (Год издания) - Автор".

 3. Создайте класс Library, который хранит массив книг и позволяет:
 • Добавлять книги.
 • Выводить список книг.
 • Искать книгу по названию.
 • Сортировать книги по году издания (новые идут первыми).
 • Сортировать книги по году рождения Автора (новые идут первыми).

4. Создайте класс с методом main. Создайте 3 автора у каждого по 2 книги. 
Добавте эти 6 книг в библиотеку. И Далее по примеру ниже.


Пример работы

Library library = new Library();

Author tolkien = new Author("Джон", "Толкин", 1892)
Book lordOfTherings = new Book("Властелин колец", tolkien, 1955)
library.addBook(lordOfTherings);

System.out.println("Все книги:");
library.printBooks();

System.out.println("Поиск книги 'Властелин колец':");
System.out.println(library.findBook("Властелин колец"));

System.out.println("Книги после сортировки по году издания:");
library.sortBooksByYear();
library.printBooks();

System.out.println("Книги после сортировки по году рождения автора:");
library.sortBooksByAuthorBirthYear();
library.printBooks();
