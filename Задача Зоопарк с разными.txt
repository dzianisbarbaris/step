Задача: Зоопарк с разными животными
Создай классы животных, которые умеют издавать звуки, двигаться и питаться.

1. Базовый класс Animal
Поля:
name (имя животного)
species (вид)
age (возраст)
Методы:
makeSound() – издает звук
move() – описывает движение
eat() – описывает, что животное ест
toString() – возвращает описание животного
2. Подклассы с разными животными
Каждое животное по-своему реализует makeSound(), move(), eat().


В main создаем один массив из 3 животных (Собака, кот, птица) 

        for (Animal animal : animals) {
            System.out.println(animal);
            animal.makeSound();
            animal.move();
            animal.eat();
            System.out.println();
        }  
Пример вывода:
Собака по имени Барсик, возраст: 3
Барсик говорит: Гав-гав!
Барсик бегает на четырех лапах.
Барсик ест собачий корм.

Кот по имени Мурка, возраст: 2
Мурка говорит: Мяу!
Мурка грациозно крадется.
Мурка ест рыбу.

Птица по имени Кеша, возраст: 1
Кеша чирикает: Чик-Чирик!
Кеша летает в небе.
Кеша клюет зерна.


*
Добавим разделение на домашних (Pet) и диких (WildAnimal) животных. 
У домашних животных будет доп. поле имя владельца и метод играть play()
У диких животных будет доп. поле среда обитания и метод охотиться.
