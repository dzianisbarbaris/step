

### Задача: Анализ списка студентов

**Описание**:  
У вас есть список студентов, каждый из которых имеет имя, возраст и средний балл. Напишите программу, которая использует Stream API для обработки этого списка и выполнения следующих требований.

**Входные данные**:  
Создайте класс `Student` с полями:
- `name` (String) — имя студента.
- `age` (int) — возраст студента.
- `grade` (double) — средний балл (от 0.0 до 100.0).

Создайте список из минимум 5 студентов с разными данными (например, вручную или с помощью генерации).

**Требования к программе**:  
Используя Stream API, выполните следующие действия:
1. Отфильтруйте студентов, которым **больше 20 лет**.
2. Отсортируйте полученный список строк по убыванию среднего балла.
3. Преобразуйте отфильтрованный список в список строк, где каждая строка имеет формат:  
   `"Студент: <имя>, балл: <grade>"`.
4. Выведите итоговый список в консоль.

**Ограничения**:
- Используйте только Stream API для обработки данных (методы `filter`, `map`, `sorted`).
- Не используйте циклы для обработки коллекций.
- Время выполнения: 30 минут.

**Пример вывода**:  
```
Студент: Анна, балл: 95.5
Студент: Иван, балл: 88.0
Студент: Мария, балл: 82.5
```
