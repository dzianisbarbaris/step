### **1. Задачи на объекты**  

#### **Задача 1: Создание и вывод объекта**  
Создайте объект `student` с полями:  
- `name` (строка)  
- `age` (число)  
- `group` (строка)  
Выведите объект в консоль.  

**Пример вывода:**  
```javascript
{ name: "Иван", age: 20, group: "ПИ-101" }
```

---

#### **Задача 2: Изменение объекта**  
Дан объект:  
```javascript
let car = {
  brand: "Toyota",
  model: "Corolla",
  year: 2020
};
```  
Измените `year` на `2022` и добавьте новое поле `color` со значением `"синий"`. Выведите обновленный объект.  

**Пример вывода:**  
```javascript
{ brand: "Toyota", model: "Corolla", year: 2022, color: "синий" }
```

---

#### **Задача 3: Подсчёт свойств**  
Напишите функцию `countProperties(obj)`, которая принимает объект и возвращает количество его свойств.  

**Пример:**  
```javascript
let user = { name: "Анна", age: 25, isAdmin: true };
console.log(countProperties(user)); // 3
```

### **2. Задачи на цикл `for`**  

#### **Задача 4: Вывод чисел от 1 до N**  
Напишите цикл `for`, который выводит числа от `1` до `N` (где `N` — заданное число).  

**Пример:**  
```javascript
let N = 5;
// Вывод: 1 2 3 4 5
```

---

#### **Задача 5: Сумма чисел в массиве**  
Дан массив чисел:  
```javascript
let numbers = [10, 20, 30, 40, 50];
```  
Используя цикл `for`, вычислите сумму всех элементов.  

**Пример вывода:**  
```javascript
Сумма: 150
```

---

#### **Задача 6: Поиск в массиве**  
Дан массив строк:  
```javascript
let fruits = ["яблоко", "банан", "апельсин", "груша"];
```  
Напишите цикл `for`, который выводит индекс элемента `"апельсин"`.  

**Пример вывода:**  
```javascript
Индекс апельсина: 2
```

### **3. Комбинированные задачи (объекты + циклы)**  

#### **Задача 7: Перебор свойств объекта**  
Дан объект:  
```javascript
let book = {
  title: "JavaScript для начинающих",
  author: "Иван Петров",
  pages: 200,
  price: 1500
};
```  
Используя цикл `for...in`, выведите все ключи и их значения в формате:  
```
title: JavaScript для начинающих  
author: Иван Петров  
pages: 200  
price: 1500  
```

---

#### **Задача 8: Фильтрация объектов в массиве**  
Дан массив объектов (студентов):  
```javascript
let students = [
  { name: "Алексей", age: 20, group: "ПИ-101" },
  { name: "Мария", age: 19, group: "ПИ-102" },
  { name: "Иван", age: 21, group: "ПИ-101" }
];
```  
Напишите цикл `for`, который выводит имена студентов из группы `"ПИ-101"`.  

**Пример вывода:**  
```
Алексей  
Иван  
```

---

#### **Задача 9: Поиск максимального значения**  
Дан объект с товарами и их ценами:  
```javascript
let products = {
  "ноутбук": 50000,
  "телефон": 30000,
  "наушники": 8000,
  "мышь": 2000
};
```  
Напишите цикл `for...in`, который находит название самого дорогого товара.  

**Пример вывода:**  
```
Самый дорогой товар: ноутбук (50000)
```
