### Задача: Сверстать расширенную карточку профиля пользователя

**Описание**:  
Создайте карточку профиля пользователя, включающую аватар, имя, возраст, email, адрес, хобби, краткое описание и кнопку "Связаться". Используйте **Flexbox** для компоновки, **position** для декоративного элемента (звезды), **float** для аватара и блочные/строчные элементы для текста и кнопки.

**Требования**:

1. **HTML-структура**:
   - Семантические теги: `<article>` (карточка), `<h2>` (имя), `<p>` (описание, возраст, email, адрес, хобби), `<button>` (действие).
   - Элементы карточки:
     - Аватар (`<img>`).
     - Имя.
     - Возраст (например, "Возраст: 25 лет").
     - Email (например, "Email: anna@example.com").
     - Адрес (например, "Адрес: Минск, ул. Ленина, 10").
     - Хобби (например, "Хобби: программирование, путешествия").
     - Краткое описание.
     - Кнопка "Связаться".
     - Декоративный значок (звезда).

2. **Позиционирование**:
   - **Flexbox**:
     - Карточка (`<article>`) — контейнер с `display: flex`, `flex-direction: column`, элементы центрированы (`align-items: center`).
     - Дополнительные поля (возраст, email, адрес, хобби) собраны в блок (например, `<div class="info">`), который использует Flexbox для горизонтального или вертикального расположения (на ваше усмотрение).
   - **Float**:
     - Аватар плавает слева (`float: left`), текст обтекает справа.
   - **Position**:
     - Звезда размещена в правом верхнем углу с `position: absolute`.
     - Карточка имеет `position: relative` для корректного позиционирования звезды.
   - **Блочные/строчные элементы**:
     - Имя, описание, кнопка — блочные элементы.
     - Поля возраст, email, адрес, хобби — блочные внутри `<div class="info">`, но могут быть строчными для компактности (например, с `display: inline-block`).

3. **Стили**:
   - Карточка: ширина 350px, рамка, тень, отступы, фон `#f9f9f9`.
   - Аватар: 80x80px, закругленный (`border-radius: 50%`).
   - Имя: крупный шрифт, центрирован.
   - Поля (возраст, email, адрес, хобби): шрифт поменьше, выровнены слева или в колонку.
   - Описание: обтекает аватар, отступы.
   - Кнопка: цветной фон, закругленные углы, hover-эффект.
   - Звезда: 20x20px, золотая, полупрозрачная.
   - Блок `.info`: отступы, возможно, фон для выделения.

4. **Адаптивность**:
   - На экранах <400px:
     - Карточка занимает всю ширину (`width: 100%`).
     - Аватар центрируется (убрать `float`, использовать `display: block`, `margin: 0 auto`).
     - Поля возраст, email, адрес, хобби — в столбец (если использовался горизонтальный Flexbox).
     - Звезда уменьшается до 15x15px.

### Расширение задачи (для интереса)

- Добавьте hover-эффект для звезды (например, `transform: scale(1.2)`).
- Сделать поля email и хобби кликабельными (например, `<a href="mailto:anna@example.com">` для email).
- Добавьте вторую кнопку (например, "Редактировать") рядом с "Связаться" с помощью Flexbox.
